# Go template

[Installation golang](https://go.dev/doc/install)

[Documentation](https://go.dev/doc/)

## Usage

```bash
go run ./cmd/cli
```

### Make (linux/darwin)

```bash
1. make // help
2. make deps // install dependencies
3. make build // build binary
```
